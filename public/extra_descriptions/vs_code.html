Hyper Key

The caps lock hyper key is included for quick left handed access to all the shortcuts.

Caps Lock Maps to left control + left shift + right command (⌃⇧⌘). The Caps lock is disabled.
Double-tap right shift Caps lock toggle

Searching

Caps Lock Hyper Key + a Quick open (search for a file)
Caps Lock Hyper Key + s Navigate symbols (search symbols)

Toggle Panels and Zen Mode

Double-tap left ctrl Toggle sidebar
Double-tap left option Toggle terminal
Right cmd Toggle Zen mode
